# 小说创作 3a.2：小说工作台项目列表与项目查看 切片规格

日期：2026-07-02
上位文档：`docs/plans/2026-06-30-novel-creation-migration-plan.md`、`docs/plans/2026-07-02-novel-generation-chain-3a-spec.md`、`docs/plans/2026-07-02-novel-inspiration-mode-3a1-spec.md`

## 定位

3a.1 之后，创作中心的「小说工作台」卡片点击后不再直接进入旧三栏编辑器，而是先进入「我的小说项目」列表，再按需进入「项目查看」或编辑器：

```text
创作中心
├─ 灵感模式 → 对话式创意流 → 蓝图/大纲 → 项目（3a.1，已落地）
└─ 小说工作台 → 我的小说项目 → 查看（项目查看页） / 创作（现有三栏编辑器）
```

**独立任务、独立提交**，不与 3a.1 混在同一提交。

## 一、我的小说项目页

风格：适配 Endless Creation 蓝紫主题、大留白、大圆角、无硬边框、微阴影、弱分割；不做传统 Web 管理后台感、不做密集表格。参考截图仅作结构参考，不照搬配色。

结构：

- 标题：我的小说项目；右上：返回（回创作中心）。
- 项目卡片列表 + 底部「创建新项目」卡（复用现有新建小说逻辑）。
- **不做假入口**：「管理后台」「导入小说文件」当前无真实能力，一律不展示（不置灰、不占位）。

项目卡片字段与推导口径（**零存储 schema 新增**，全部运行时推导）：

- 小说标题、最近更新时间（既有字段）。
- 状态标签：`蓝图` = 无任何有正文的章节；`连载中` = 至少一章有正文。**不做「已完成」**——当前无完结字段，不发明。
- 完成度 = 有正文章节数 / 总章节数（总章节为 0 时显示 0% 或空态）；进度条同源。
- 按钮：查看 → 项目查看页；创作 → 现有三栏编辑器。

跨层小改动（本切片唯一超出 renderer 的改动）：`NovelSummary` 增加计算字段 `filledChapterCount`（列表时统计 `content` 非空的章节数）。这是**派生值不是存储字段**，`novel.json` 结构不变；`electron/main` 的 `toNovelSummary` 与 `rendererBridge` web fallback 同步补齐。

## 二、项目查看页

顶部栏：左侧小说名 + 最近更新；右侧「返回列表」「开始创作」（进现有编辑器）。

左侧「蓝图导航」，**第一版只放 3 个真实项**（2026-07-02 老板拍板，与「不做假入口」原则一致；能力落地一个加一项，不做占位页、不做置灰预告）：

1. 项目概览 — 定位与整体梗概
2. 章节大纲 — 故事结构规划
3. 章节内容 — 章节与正文概要

世界设定 / 主要角色 / 人物关系 / 情感曲线 / 伏笔管理 **不进导航**，等对应阶段（四/五）能力落地再加。

## 三、项目概览页

- 核心摘要：优先 `blueprint`，无则依次回退 `summary` / `idea`；全空显示优雅空态。
- 目标受众 / 类型 / 风格 / 基调：**不新增结构化字段**；blueprint 有内容时作为普通文本展示，无则空态。
- 按钮：编辑（就地编辑 blueprint / summary / idea，走现有 `updateNovel` + autosave）、开始创作。

## 四、章节大纲页

- 列表展示 `chapters`：序号（按 `order` 排序的显示序号）、标题、`outline` 内容。
- 支持新增章节、编辑大纲（复用现有章节增改逻辑）。
- **不改 `Chapter.order`、不引入 `chapter.index`。**

## 五、章节内容页

- 章节列表 + 当前章节：标题、字数、`outline`（计划大纲）、`content` 概要。
- 点击「开始创作」或具体章节 → 进入现有三栏编辑器定位到该章；**不重写编辑器**。
- **不照搬截图中的**：版本 tab / 评审 tab（3b、阶段四）、导出 TXT（阶段五）、「已完成」徽章（无字段）。

## 防跑偏约束

本阶段**严禁**实现：

- 世界观 / 角色卡 / 人物关系的结构化 CRUD 或占位页
- 情感曲线、伏笔追踪（含任何伏笔 schema）
- 多版本、RAG、Prompt 后台、自动生成整本书
- 封面（卡片不做封面位，或仅纯色装饰，不做上传/生成）
- 除 `NovelSummary.filledChapterCount` 外的任何类型/存储/IPC 变更

## 验收标准

1. 点击「小说工作台」进入「我的小说项目」页，项目卡片展示已有小说，状态与完成度按推导口径显示。
2. 「创建新项目」可用，复用现有新建逻辑。
3. 「查看」进入项目查看页，蓝图导航仅 3 项。
4. 项目概览按 blueprint → summary → idea 回退展示，可就地编辑并自动保存。
5. 章节大纲页展示并可编辑 `outline`，可新增章节；排序与删除重排不回归。
6. 章节内容页展示标题 / 字数 / outline / content 概要；「开始创作」与点击章节能进入现有编辑器并定位到对应章节。
7. 无数据处（无小说、无章节、无 blueprint/outline/content）均为优雅空态，不崩溃。
8. 不影响灵感模式（3a.1）、按大纲生成正文（3a）、小说 CRUD / 保存恢复 / 续写润色改写、生图与资产模块。
